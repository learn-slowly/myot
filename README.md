# 내옷 myot

워크웨어 · 아이비 · 캐주얼 코디 앱

## 기능
- **옷장**: 시즌별 필터로 전체 아이템 확인
- **코디**: 하의 선택 → 어울리는 상의/아우터/신발 추천
- **오늘 뭐 입지**: 기분 + 시즌으로 랜덤 코디 제안
- **찜**: 구매 예정/찜 목록 관리 + 저장한 코디

## 기술 스택
- Next.js 15 + TypeScript + Tailwind CSS v4
- localStorage로 데이터 persist
- PWA 지원 (홈화면 추가 가능)

## 개발
```bash
npm install
npm run dev
```
