# 내옷 myot — My Only Trend

> AI 기반 개인 스타일 옷장 관리 플랫폼 "유행은 따르지 않는다. 나만의 트렌드를 입는다."

## 뭐하는 앱?

옷은 많은데 입을 게 없다면, 옷이 문제가 아니라 **조합**이 문제다.

myot은 **자기만의 스타일 규칙**이 있는 사람을 위한 AI 옷장 관리 앱이다. 범용 AI 추천이 아니라, 사용자 개인의 옷장 데이터와 스타일 원칙을 학습해서 진짜 입을 조합을 찾아준다. 시즌별 소재 분리, 개인 스타일 축 기반 추천, 구매 판단까지.

## 주요 기능

### 📸 OOTD — 오늘 뭐 입었어?

사진 올리면 AI가 착장 분석. 내 옷장 아이템과 자동 매칭 후 수정 가능. 저장하면 AI가 코디 코멘트까지 달아줌. 착용 통계로 자주 입는 옷, 안 입는 옷 파악.

### 👔 옷장 — 내 옷 전부

카테고리별 보기, 시즌 필터, 아이템 추가/편집/삭제. 사진 등록하면 썸네일로 표시. 새 아이템 추가하면 AI가 기존 옷장과 비교해서 코디 조합 자동 생성.

### 🎨 코디 — 이 옷에 뭘 입지?

하의, 상의, 아우터 중 아무거나 먼저 선택하면 어울리는 나머지를 추천. 선택할수록 좁혀짐. 시즌별 완전 분리로 소재 혼재 없음.

### ☀ 오늘 뭐 입지 — 지금 나가는데?

현재 위치 날씨 자동 연동. 기온으로 시즌 자동 판단, 비 오면 적합한 신발 우선 추천, 최근에 안 입은 옷 우선 추천.

### 🤔 살/말 — 이거 살까?

사고 싶은 옷 사진 올리면 내 옷장 데이터 기반으로 AI가 구매 판단. 중복 체크, 활용도, 스타일 적합성, 색상 조화를 분석해서 🟢살 / 🟡고민 / 🔴말 판정. "안 입는 옷 사지 않기"를 데이터로 실천.

### ⭐ 찜 — 사고 싶은 것들

가격, 구매처 링크, 사진, 메모 관리. 상태 카테고리 자유 생성 가능 (다음달 구매, 여름에 찾기, 해외직구 대기 등 원하는 대로).

## 차별화

| 기존 옷장 앱 | myot |
| --- | --- |
| 범용 AI 추천 | 사용자 개인 옷장 데이터 기반 추천 |
| 시즌 필터만 제공 | 시즌별 소재 완전 분리 (울 니트가 여름에 추천되지 않음) |
| 옷 하나하나 등록해야 함 | OOTD 찍으면서 자연스럽게 옷장이 채워짐 |
| 통계만 보여줌 | 착용 데이터 → 구매/처분 판단까지 연결 |
| 추천만 함 | "살까 말까" AI 판단으로 불필요한 소비 방지 |

## 기술 스택

| 영역 | 기술 |
| --- | --- |
| 프레임워크 | Next.js 15 + TypeScript + Tailwind CSS v4 |
| DB | Supabase PostgreSQL |
| 이미지 | Cloudinary CDN |
| AI | Anthropic Claude Sonnet (서버사이드 API Route) |
| 날씨 | Open-Meteo (무료, 키 불필요, 상용 가능) |
| 배포 | Vercel (GitHub 자동 배포) |
| PWA | 홈 화면 추가 가능 |

## 개발

```bash
# 의존성 설치
npm install

# 환경변수 설정 (.env.local)
ANTHROPIC_API_KEY=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=

# 로컬 개발
npm run dev
```
